A command-line version of the classic Hangman game with a database of words based on popular TV shows and movies.

Hangman Terminal App

Working in groups of two we were tasked with creating a Ruby terminal application. With free choice of developing any type of application, my partner and I opted to create Hangman.

This choice was made as we felt that a terminal game had a good balance of complexity and difficulty that was also reasonable for the given time frame of the project. We also just thought it would be cool and enjoyable to create an interactive game.

Since the user is only interacting with the command prompt and reading text we wanted to add another element to the game and make to make it more interesting.

We achieved this by creating and implementing sound effects that would play during key moments of the game such as game launch, choosing a category and guessing a letter.

Technology Used

The whole application was developed using Ruby with the use of the Faker gem to create the database of words.

GitHub was also used for version control and collaboration.
